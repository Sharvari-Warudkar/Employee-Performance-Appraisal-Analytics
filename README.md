# Employee Performance & Appraisal Analytics System

## Project Overview

This project analyzes employee performance, salary, project allocation, and appraisal data for ScienceQtech. The objective is to support HR decision-making by generating insights on employee ratings, salary distribution, bonuses, experience levels, and project assignments using SQL.

## Business Problem

ScienceQtech's HR department needed a reporting system to evaluate employee performance, identify top performers, calculate bonuses, validate job profiles, and generate appraisal reports for annual performance reviews.

## Dataset

The project uses three datasets:

### Employee Records
Contains employee details such as:
- Employee ID
- Name
- Department
- Salary
- Experience
- Rating
- Manager ID
- Project ID

### Project Details
Contains:
- Project Name
- Domain
- Status
- Development Quarter

### Data Science Team
Contains employee details for data science team members.

## Analysis Performed

- Employee Performance Analysis
- Salary Distribution Analysis
- Bonus Calculation
- Department Analysis
- Experience Ranking
- Employee-Manager Mapping
- Project Assignment Tracking
- Job Profile Validation

## Key Findings

- Identified top-performing employees based on appraisal ratings.
- Automated employee bonus calculation.
- Ranked employees based on experience.
- Evaluated department-wise employee distribution.
- Validated employee job roles using years of experience.
- Tracked project allocation and project status.

## Business Impact

- Reduced manual reporting effort.
- Improved appraisal transparency.
- Enabled data-driven HR decision making.
- Automated performance analysis and employee evaluation.
